# Numberle
Wordle but with Numbers.

## Basic concept for the game:

1. The game generates a random X-digit number (let's say 4 digits to start with).
2. The player has Y attempts to guess the number correctly.
3. After each guess, the game provides feedback:
  - Correct digit in the correct position (analogous to Wordle's green)
  - Correct digit but in the wrong position (analogous to Wordle's yellow)
  - Digit not in the number at all (analogous to Wordle's gray)
